Factory overview

Welcome to Joshfire factory developer site!

The factory focuses on easy cross-device "content-based" application packaging and deployment. Factory users benefit from template developers to create Web applications without having to worry about coding and adapting the experience to multiple devices. Using the factory does not require any specific coding skills. The factory targets the public at large. In turn, the factory allows developers to sell templates and services.

Through the factory, users:

  1. select datasources that they wish to expose (RSS feeds, pictures, videos, Twitter feeds, etc.);
  2. apply templates that control the visual layout and functionalities of the Web application;
  3. install optional plugins to add extended features;
  4. package and deploy resulting Web applications.

Template developers:

  1. create templates that address specific user scenarios;
  2. publish templates on the factory for users;
  3. sell templates, support or services based on the templates they create.

This site targets templates developers. The documentation assumes that the reader has basic skills in usual Web acronyms such as HTML, CSS, JavaScript, JSON.
