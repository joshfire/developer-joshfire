Creating an application

The main tabs are the different steps to follow to create an application. To learn more about any of them, go to their dedication documentation page:

  1. Template

    Choosing a template means choosing the visual style and the main features of your application. A template can be applied to one or more device family.

  2. Data

    The content displayed in your application comes from various data sources.

  3. Preview

    On the right side of the screen, you can see a ribbon, click on this ribbon to open the Preview window.

  4. Add-ons

    An add-on enhances your app with a specific feature.

  5. Configuration

    Global configuration of your app, such as the application title or icon, is done here. Configuration of selected templates and add-ons is also done here.

  6. Pay and Deploy

    Build and deploy your application to the web or application stores.
