Template configuration

You can access the template configuration page by selecting the Configuration tab and then clicking on the template name in the left menu.

Configuring a template

The template configuration page allows you to change parameters specific to this template.

For most template you will be able change which data you want to add to it:

Select from your previously configured data sources or create a new one directly from this page by selecting "Add a new datasource":

If you previously configured data sources for your app, they may be already inserted automatically in a data-source field. If the data source field is a multiple data source field, you can add an additional data source by clicking the "add more" link. Clicking the "delete last" link will remove the last data source of the list.
