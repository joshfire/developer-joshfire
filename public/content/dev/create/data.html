Data

While a template determine the look and the functionnalities of your application, you still have to add content to it. The data section is here to let you select various external sources of data that will be displayed inside your application.
Joshfire created simple adapters for some sources of data.

Available data-sources are the big icons displayed on this page. To create a new data-source for your application, click on the its icon.
On the left of the page, you can find the list of the data-sources you already configured. Click an item of this list to update the configuration of this already created data-source.

If you need help configuring a particular data-source, please, read more about it in the dedicated page:

  1. Youtube
  2. Twitter
  3. Flickr
  4. Static Post

If you do not find the data source you look for, you can contact Joshfire or visit the datasource development section
