Template detail page

The template detail page displays a description and screenshots on supported device of the template. By reading the description, you can better understand for which usage this template has been created. Screenshots are sorted by device types and are screenshots of examples of applications built using this template, the content that you see in these screenshots will be replaced by your content

The box on the right is important: You can select to which device you want to apply this template. Devices that are grayed are devices not supported by the template. By checking a device, and clicking "Install", you will install this template on the selected device. If you previously configured another template on this device, it will be replaced by this new template.

The template price depends on the devices you selected. When you pay for a template you can use it in your application. You get future updates of this template for free.
