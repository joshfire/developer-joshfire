Style Guidelines

JavaScript

We follow the Google JavaScript Style Guide. It might also be a good idea to do the same in your template code.

Most important things:

  • 2-space indentation,
  • always start your curly braces on the same line as whatever they're opening,
  • always use semicolons.

HTML

  • 2-space indentation,
  • all attributes should be quoted (with double quote).

CSS

  • 2-space indentation.
  • always start your curly braces on the same line as whatever they're opening.
