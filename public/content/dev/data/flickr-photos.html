Flickr data source

Description

The Flickr data source returns a list of pictures hosted on the Flickr website.

Configuration

Search by text

Enter keywords to perform a searh on Flickr. Most recent pictures are returned first. You can also use advanced search querries such as Joshfire OR factory, using OR will search pictures with any of the given words.
The results are the same than on the Flick Search page, sorted with the "Recent" option.

Load a photo set

We strongly advise you to create your own set to import Flickr pictures into your application, you will keep a total control on which pictures are displayed.

The simplest way to configure a photo set is to copy and paste a photo set URL. Just go on a set page, such as this one, and copy its web adress. It should look like this: http://www.flickr.com/photos/YOURUSERNAME/sets/SETID/.

You can find your existing sets by clicking on the "You" menu then selecting "Your sets". To create a new set, you need first to upload pictures and then go to the set organizer (accessible from the "Organize & Create", and "Your sets" menu).
If you need more help using Flickr sets, you can visit the Set help page.

Authentication settings

You can preview your application without entering your Flickr API key, however, to deploy it, you need to submit your Flickr API key. To obtain an API key, follow the instructions on Flickr's API Web site.
If you already have an API key that you would like to use, you can find it in the Your account, Sharing & Extending. However, we advise to create an API key per generated application.
