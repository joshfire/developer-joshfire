Twitter data source

Description

The Twitter data source returns a list of tweets. Tweets are compatible with the majority of the available templates since they look like small articles.

Configuration

Search by text

Searching by text will return the result of a twitter search. You will receive the same results that you get by using https://twitter.com/search.

You can enter simple words, hastags (such as #JoshfireFactory) or even advanced queries (such as JoshfireFactory OR "Joshfire Factory"). Use twitter's advanced search page to get help about generating an advanced query.

Search by username

Searching by username will return the tweets written by the given user. Simply enter a twitter username such as @joshfire.

Exclude retweets

Checking this box will remove retweets from the results. Retweets are tweets that starts with "RT" and a username, for example: "RT @JoshfireFactory: Discover the #JoshfireFactory http://factory.joshfire.com"

Exclude replies

Checking this box will remove replies from the results. Replies are tweets that starts with a username, for example: "@JoshfireFactory Where is the documentation?".
