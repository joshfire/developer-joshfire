Static Post data source

Description

The "Static Post" datasource lets you write a custom HTML text, by using an integrated text editor or by writting directly the HTML code.
This data source is "static", this means that it doesn't fetch online content from web services: its content is created from within the Factory.
It can be useful to add "about pages" to your application, for example to give contact details about your business.

Configuration

Use our simple integrated text editor to create your content from the data source configuration page. You can also manually edit the HTML code. We advise you to keep it as simple and clean as possible. Adding custom styles or tables may break the layout of the application or not integrate well with its style.

Update

The content of such "static data source" will be shipped inside your application when you deploy it. This means that even if you edit it from the Factory, the content is then written in your application and cannot be changed without re-publishing it.
